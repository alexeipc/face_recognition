# Điểm danh bằng nhận diện khuôn mặt

### Cần có

* Python 3.6.4 
* OpenCV 3.4.1 or opencv-contrib-python 4.0
* Numpy
* Pillow
* SQLite

## Cách chạy chương trình

* Chạm main.py bằng câu lệnh:
```bash
python main.py
```
* Cho máy học khuôn mặt bằng câu lệnh:
```bash
>> insert
```
* Chạy chương trình điểm danh bằng câu lệnh:
```bash
>> start
```
